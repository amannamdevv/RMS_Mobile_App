Privacy Policy for RMS

Effective Date: April 15, 2026

Developed by: Shroti Telecom Pvt. Ltd.

1. Introduction

Shroti Telecom Pvt. Ltd. ("we", "our", or "us") operates the RMS Pro mobile application (the "App"). This Privacy Policy explains how we collect, use, store, share, and protect information when you use our App. By using RMS Pro, you agree to the terms of this Privacy Policy.

This App is intended solely for authorized enterprise users — specifically telecom field operators, NOC engineers, and site managers employed by or contracted with Shroti Telecom Pvt. Ltd. or its clients.

2. Information We Collect

a) Information You Provide:

  • Mobile phone number (used for OTP-based login authentication)
  • Full name (stored locally on device after login)

b) Information Collected Automatically:

  • Session authentication tokens (stored in device AsyncStorage)
  • App usage data (screen navigation, API calls) — used solely for operational functionality

c) Information We Do NOT Collect:

We do not collect precise GPS location data, device contacts, camera images (unless user-initiated), biometric data, or advertising identifiers. We do not use third-party analytics SDKs.

3. How We Use the Information

We use the information collected solely for:

  • Authentication: Verifying user identity via OTP before granting access to site data.
  • Personalization: Displaying the logged-in user's name in the app interface.
  • Operational Function: Fetching and displaying telecom site monitoring data from our backend API servers.
  • File Sharing: Enabling users to export and share reports via device sharing mechanisms.

4. Data Storage & Security

Local Storage: User authentication tokens and the user's full name are stored locally on the device using React Native AsyncStorage. This data is encrypted at the OS level.

Server Storage: All site monitoring data (alarms, energy readings, uptime metrics) is stored on Shroti Telecom Pvt. Ltd.'s backend servers.

Data Security: All API communications are conducted over HTTPS (TLS encryption). Authentication uses OTP-based two-factor verification.

5. Data Sharing

We do not sell, rent, or trade your personal information to any third parties. Data is only shared within the organization for authorized operational purposes or if required by law (Legal Compliance).

6. Children's Privacy

RMS Pro is strictly a professional enterprise application intended for adults aged 18 and above. We do not knowingly collect personal information from individuals under the age of 18.

7. Your Rights

As a user of RMS Pro, you have the right to access, correct, or request deletion of data associated with your account. You can stop using the App at any time; session data will be removed upon logout.
