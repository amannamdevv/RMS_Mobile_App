Privacy Policy for RMS

Effective Date: April 15, 2026

Last Updated: April 15, 2026

Developed by: Shroti Telecom Pvt. Ltd.

1. Introduction

Shroti Telecom Pvt. Ltd. ("we", "our", or "us") operates the RMS mobile application (the "App"). This Privacy Policy explains how we collect, use, store, share, and protect information when you use our App. By using RMS, you agree to the terms of this Privacy Policy.

This App is intended solely for authorized enterprise users — specifically telecom field operators, NOC engineers, and site managers employed by or contracted with Shroti Telecom Pvt. Ltd. or its clients.

2. Information We Collect

a) Information You Provide:

  • Mobile phone number (used for OTP-based login authentication)
  • Full name (stored locally on device after login)

b) Information Collected Automatically:

  • Session authentication tokens (stored in device AsyncStorage)
  • App usage data (screen navigation, API calls) — used solely for operational functionality

c) Information We Do NOT Collect:

  • We do not collect precise GPS location data
  • We do not collect contacts
  • We do not collect camera or microphone data
  • We do not collect biometric data
  • We do not use advertising identifiers or third-party analytics SDKs

3. How We Use the Information

  • Authentication: Verifying user identity via OTP
  • Personalization: Displaying the logged-in user's name
  • Operational Function: Displaying telecom site monitoring data
  • File Sharing: Enabling report export and sharing

4. Data Storage & Security

Local Storage: User authentication tokens and name are stored using secure device storage (AsyncStorage).

Server Storage: Site data is stored on Shroti Telecom Pvt. Ltd.'s backend servers.

Security: All communication is secured using HTTPS (TLS encryption).

5. Data Sharing

We do not sell, rent, or share personal data with third parties. Data may only be shared within the organization or if required by law.

6. Children's Privacy

This app is intended for users aged 18 and above. We do not knowingly collect data from children.

7. Your Rights

  • You can request access to your data
  • You can request correction or deletion
  • You can stop using the app anytime
